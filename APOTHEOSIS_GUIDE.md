# Apotheosis 모드 설명 및 사용법

## 개요

**Apotheosis**는 이 마인크래프트 모드팩 RPG의 **핵심 장비 강화 시스템**을 담당하는 모드입니다. 바닐라 인챈팅 시스템을 완전히 오버홀하고, 리포징(Reforging)과 젬 소켓팅(Gem Socketing) 시스템을 추가하여 장비를 극한까지 강화할 수 있게 해줍니다.

---

## Apotheosis 3대 축

Apotheosis는 세 가지 핵심 강화 시스템으로 구성됩니다:

### 1. 인챈팅 (Enchanting)

- **바닐라 인챈팅 오버홀** — 수백 개의 추가 인챈트 제공
- 대표 인챈트: `Fortune IX`, `Boon of the Earth VI` 등
- 무기, 도구, 방어구 모두에 대폭 강화된 인챈트 적용 가능
- **필요 시설**: 인챈트 테이블 + 책장

### 2. 리포징 (Reforging)

장비에 추가 스탯, 효과, 젬 소켓을 부여하는 시스템입니다.

**5단계 등급 체계:**

| 등급 | 설명 |
|------|------|
| Common | 기본 등급 |
| Uncommon | 약간의 추가 효과 |
| Rare | 중급 효과 + 소켓 |
| Epic | 고급 효과 + 다수 소켓 |
| **Mythic** | 최고 등급 — 4 보너스, 3 효과, 3 소켓 |

- **필요 시설**: Reforging Table

### 3. 젬 소켓팅 (Gem Socketing)

리포징이나 Sigil로 열린 소켓에 젬을 장착하여 빌드에 맞는 스탯을 커스텀합니다.

**6단계 젬 품질:**

```
Cracked → Chipped → Flawed → Normal → Flawless → Perfect
```

**핵심 젬 목록:**

| 젬 | 효과 (Perfect 기준) | 용도 |
|----|---------------------|------|
| Warlord | 크리티컬 데미지 +40% | 물리 딜러 |
| Samurai | 크리티컬 확률 +40% | 물리 딜러 |
| Spell Level | 주문 레벨 +1 | 마법사 (Mage) |
| Earth / Ballast | 방어/체력 특화 | 탱커 |

---

## 소켓 최대화 순서

장비의 소켓 수를 극대화하려면 아래 순서를 따르세요:

```
1. Superior Sigil 사용  →  최대 4 소켓 추가
2. 리포징 (Mythic)      →  3 소켓 추가
3. 스킬트리 (Jeweler)   →  +1 소켓 추가
```

---

## 필요 제작 시설

| 시설 | 용도 |
|------|------|
| 인챈트 테이블 + 책장 | 인챈팅 |
| Reforging Table | 리포징 |
| Salvaging Table | 장비 분해 |
| Chronicle of Shadows | 리포징 재료 |
| Sigil (Socketing / Superior) | 소켓 추가 |

## 진행 타임라인에서의 Apotheosis

### 초반 (Early)
- **Champions Trials** 퀘스트를 시작하면서 Apotheosis 재료를 수집합니다.
- 이 단계에서는 아직 Apotheosis 시설을 본격적으로 구축하지 않습니다.

### 중반 (Mid) — Step 03
- **Apotheosis 강화 인프라를 구축**하는 핵심 단계입니다:
  1. 인챈트 테이블 + 책장 설치
  2. Reforging Table 제작
  3. 젬 수집 시작
  4. End Eyes 병행 수집 (15종 중 12개 필요)
- **Aquaflora** 등 무기를 Apotheosis 인챈팅으로 강화

### 중후반 (Mid-Late) — Step 05
- **풀 강화 스택 완성**:
  - 인챈트 + 리포징 + 젬 소켓팅 + 스킬트리 모두 풀업
  - Perfect 젬 획득 및 장착
- T2 보스 도전 전 필수 완료

### 엔드게임 (Endgame)
- Mythic 리포징으로 최대 보너스 확보
- 완벽한 젬 조합으로 빌드 최적화
- T3 보스 및 최종 콘텐츠 도전

---

## 방어 공식

Apotheosis의 방어 시스템은 다음 공식을 따릅니다:

```
피해감소 = A / (A + X)
```

- **A** = 받는 피해 / 2 (최소 10)
- **X** = 아머 수치
- **Protection** 인챈트 1포인트 = 2.5% 감소 (최대 85%)

---

## 강화 권장 순서 (요약)

```
① 인챈팅 — 인챈트 테이블에서 최고 등급 인챈트 부여
② Sigil 사용 — Superior Sigil로 소켓 최대 확보
③ 리포징 — Mythic 등급까지 리포징하여 추가 소켓 + 스탯
④ 젬 장착 — 빌드에 맞는 Perfect 젬 소켓팅
⑤ 스킬트리 — Jeweler 노드로 +1 소켓 추가
```

## 팁

- **Silverfish 파밍**으로 젬을 효율적으로 수집할 수 있습니다.
- Sigil을 먼저 사용한 후 리포징하면 소켓 수가 극대화됩니다 (순서 중요!).
- Mythic 리포징은 여러 번 시도가 필요할 수 있으므로 재료를 넉넉히 준비하세요.
- 빌드에 따라 젬 조합이 달라지므로, 물리/마법/탱커 중 자신의 빌드를 먼저 결정하세요.
