# Korean Instagram Photography Reference

Claude MUST study this reference and apply scene-appropriate techniques
when generating any photo-style image. These are real-world shooting
patterns observed from popular Korean Instagram accounts.

---

## 1. Scene-Specific Shooting Patterns

### 1.1 카페 음식 (Cafe Food)
```
구도:
  - 탑뷰 (overhead flat lay): 접시 전체 + 음료 + 소품
    · 폰을 테이블에 완전 평행하게
    · 접시를 중앙 또는 약간 오프센터
    · 주변에 수저, 냅킨, 작은 소품 자연스럽게 배치
    · 아이템 사이에 여백(negative space) 충분히
  - 45도: 음식의 높이/볼륨감을 보여줄 때
    · 케이크, 크루아상 같은 입체적 음식
    · 테이블 가장자리가 살짝 보임
    · 배경에 카페 인테리어 흐리게
  - 사이드뷰 (eye-level): 레이어드 음료, 층이 보이는 디저트
    · 그릇/잔 옆에서 수평으로
    · 배경 깊이감 최대화

조명:
  - 창가석 측광이 기본 (왼쪽 또는 오른쪽에서)
  - 직사광선보다 커튼 너머 확산광 선호
  - 음식 위에 부드러운 하이라이트, 반대편에 은은한 그림자
  - 절대 플래시 사용 안 함

색감:
  - 뮤트 웜: 채도 낮추고 따뜻하게
  - 흰색은 크리미하게 (순백 X)
  - 음식 색은 자연스럽게 유지 (과채도 X)
  - 나무/대리석 테이블 색은 약간 바래게

소품:
  - 미니멀: 접시 1-2개 + 음료 + 수저 정도
  - 꽃, 엽서, 책 등은 모서리에 살짝
  - 브랜드 로고 보이는 컵이나 접시 (카페 정체성)
```

### 1.2 인물/셀카 (Portrait/Selfie)
```
구도:
  - 3/4 얼굴: 한쪽으로 약간 돌린 얼굴, 자연스러운 미소
  - 셀카: 팔 길이 거리, 눈높이보다 살짝 위에서
  - 전신: 거울샷 또는 동행이 찍어줌, 머리 위 여백 적당히
  - 뒷모습: 걸어가는 뒷모습, 벚꽃/단풍 길

조명:
  - 창가 자연광이 기본 (한쪽 밝고 한쪽 약간 어둡)
  - 카페 실내: 따뜻한 조명 + 창가 혼합
  - 야외: 흐린날 > 맑은날 (그림자 적으니까)
  - 골든아워: 일출 직후 / 일몰 직전

피부톤:
  - 밝고 깨끗하지만 인위적이지 않게
  - 약간의 피부결 보이게 (과보정 X)
  - 따뜻한 베이스에 볼/코에 자연 홍조
  - 눈 밑에 살짝 생기 있는 그림자 (리얼함)

포즈:
  - 힘들이지 않은 자연스러움이 핵심
  - 턱을 약간 당기고 눈은 살짝 내려
  - 머리카락 쓸어올리는 중간 동작
  - 음료 들고 마시려는 찰나
```

### 1.3 풍경/거리 (Landscape/Street)
```
구도:
  - 길/터널: 중앙 소실점 구도, 양옆에 벚꽃/단풍 나무
  - 한강: 수평선 1/3 지점, 하늘 넓게
  - 골목: 세로 구도, 좁은 길 양옆 건물
  - 야경: 네온/가로등 보케 전경에, 거리 뒤로

조명 & 시간대:
  - 새벽/이른 아침: 맑은 공기, 차가운 파스텔
  - 골든아워: 따뜻한 주황, 긴 그림자
  - 블루아워: 보라/파랑 그라데이션 하늘
  - 야간: 네온 색감, 젖은 도로 반사

색감:
  - 벚꽃: 파스텔 핑크 + 크리미 화이트 + 하늘 연파랑
  - 단풍: 오렌지 + 레드 + 골드 + 갈색
  - 여름: 초록 + 하늘파랑 + 흰 구름
  - 겨울: 모노크롬에 가까운 뮤트, 차가운 톤
```

### 1.4 반려동물 (Pets)
```
구도:
  - 눈높이 (eye-level): 바닥에 엎드려 동물 눈높이에서
    · 가장 친밀하고 귀여운 느낌
    · 코/눈에 초점, 귀/뒷배경 보케
  - 위에서 내려다봄: 집사가 서서 내려다본 앵글
    · 동물이 올려다보는 눈
    · 바닥/베딩 재질이 잘 보임
  - 자는 모습: 측면에서 가까이, 발바닥/코 강조
  - 하품/늘어짐: 순간 포착 느낌

조명:
  - 실내 형광등/LED: 가장 흔함, 평평한 조명
  - 창가 자연광: 한쪽 밝음, 이상적이지만 항상 가능하진 않음
  - 폰 플래시: 사용 안 함 (빨간 눈, 부자연스러움)

촬영 장비:
  - 대부분 스마트폰 (iPhone/Galaxy)
  - 인물사진 모드(portrait mode) 배경흐림 가끔 사용
  - 의도적 약간의 흔들림/불완전함 = 진짜 느낌
  - 과도한 리터치 없음

색감:
  - 따뜻하고 부드러운 톤
  - 블러시 핑크, 크림, 라벤더, 따뜻한 베이지
  - 동물 털색은 자연스럽게 유지
  - 배경은 약간 바래고 부드럽게
```

### 1.5 제품/소품 (Product/Object)
```
구도:
  - 탑뷰 플랫레이: 제품 + 관련 소품 2-3개
    · 대리석/나무/린넨 배경
    · 여백 충분히, 깔끔하게
  - 45도: 제품의 형태감
  - 손에 든: 크기감 전달, 사용 중 느낌

소품:
  - 유칼립투스 잎, 말린 꽃
  - 크래프트 페이퍼, 린넨 천
  - 커피잔, 안경, 책
  - 모든 소품은 같은 톤으로 통일
```

## 2. Korean Instagram Color Grading Presets

### 2.1 웜 뮤트 (Warm Muted) — 가장 인기
```
Lightroom/VSCO 세팅 느낌:
  온도(Temperature): +5~10 (약간 따뜻하게)
  색조(Tint): +3~5 (약간 마젠타)
  노출(Exposure): +0.1~0.3 (약간 밝게)
  대비(Contrast): -10~-20 (대비 낮춤)
  하이라이트(Highlights): -20~-40 (밝은 부분 억제)
  그림자(Shadows): +20~+40 (어두운 부분 올림 = lifted blacks)
  흰색(Whites): -10
  검정(Blacks): +10~+20 (순수 검정 없앰)
  선명도(Clarity): -5~-10 (약간 소프트)
  채도(Saturation): -15~-25 (전체 채도 내림)
  활기(Vibrance): +5~+10 (주요 색만 살짝 올림)

결과: 따뜻하면서 바랜 느낌, 순수한 흰/검은이 없는 소프트한 톤
프롬프트: "muted warm tones, lowered contrast, lifted blacks,
          reduced saturation, slightly warm temperature shift,
          soft clarity, faded film look"
```

### 2.2 쿨 파스텔 (Cool Pastel) — 봄/벚꽃
```
  온도: -3~-5 (약간 차갑게)
  색조: +5~+10 (핑크 방향)
  대비: -15~-25
  하이라이트: -30 (부드러운 흰색)
  그림자: +30 (밝은 그림자)
  채도: -20~-30 (파스텔화)
  핑크/마젠타: +15 (벚꽃 강조)

결과: 연분홍, 라벤더, 하늘색 파스텔, 몽환적
프롬프트: "cool pastel tones, pink and lavender tint,
          very low contrast, dreamy soft, airy, light"
```

### 2.3 어스 톤 (Earth Tone) — 가을/카페
```
  온도: +10~+15 (따뜻하게)
  채도: -20 (전체 내림)
  오렌지/갈색: +20 (강조)
  초록: -30 (녹색 억제)
  블루: -20 (파란 억제)

결과: 갈색, 베이지, 머스타드, 오렌지 계열 톤
프롬프트: "earthy warm tones, brown beige mustard palette,
          greens desaturated, blues muted, autumn warmth"
```

### 2.4 클린 화이트 (Clean White) — 미니멀/제품
```
  노출: +0.3~+0.5 (밝게)
  대비: -10
  하이라이트: +10 (밝은 부분 더 밝게)
  흰색: +20
  채도: -10 (전체 살짝 내림)
  선명도: +5 (약간 선명)

결과: 밝고 깨끗하고 미니멀, 화이트가 지배적
프롬프트: "bright clean white dominant, minimal,
          slightly overexposed highlights, low saturation,
          airy and fresh, Scandinavian clean feel"
```

### 2.5 무드 다크 (Moody Dark) — 야간/바/레스토랑
```
  노출: -0.3~-0.5 (어둡게)
  대비: +10~+15 (대비 올림)
  하이라이트: -40 (밝은 부분 억제)
  그림자: -10 (그림자 더 깊게)
  온도: +5 (따뜻한 어둠)
  채도: -15

결과: 어둡지만 따뜻한, 무디한 분위기, 양초/조명 강조
프롬프트: "moody dark warm tones, deep shadows, suppressed highlights,
          warm darkness, candlelight or ambient only,
          intimate atmospheric"
```

## 3. Common Mistakes to Avoid

```
❌ 과채도 (Oversaturation)
  한국 인스타는 채도를 줄이는 게 기본. "vibrant" "saturated" 쓰지 말 것

❌ 순수한 흰색/검정 (Pure white/black)
  lifted blacks + suppressed highlights = 순수 흰/검 없음

❌ 과보정 (Over-retouching)
  피부 질감 완전 제거, 배경 완전 제거 등은 인스타 감성이 아님

❌ 스튜디오 컨셉샷 느낌
  "award winning" "gallery quality" "professional large-format"
  이런 키워드는 캐주얼 인스타 감성과 반대

❌ 대칭/완벽한 구도
  약간의 비대칭, 잘림(cropping), 기울어짐이 더 자연스러움

❌ 선명도 과다 (Over-sharpening)
  clarity/sharpness를 과하게 올리면 인스타 감성 파괴
  약간 소프트한 게 맞음
```
